# Randy Pong

A Pong clone where the ball is a man named Randy.

Randy flails his arms, spins through the air, and gets smacked back and forth until someone scores. First to 7 wins.

## Play

Open `index.html` in a browser, or serve the folder locally:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080

## Controls

| Key | Action |
|-----|--------|
| `W` / `↑` | Move paddle up |
| `S` / `↓` | Move paddle down |
| `Space` | Serve Randy (or restart after a match) |

## Stack

- HTML5 Canvas
- Vanilla JavaScript
- No dependencies
